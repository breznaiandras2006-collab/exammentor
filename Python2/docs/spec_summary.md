# ExamMentor – Spec summary (for future reference)

Rövid összefoglaló a feltöltött PDF-ből: „Tanulást segítő alkalmazás: PDF-kezelés, AI-funkciók, szövegszerkesztés és PPT generálás”.

## 1) UX / termék
- Profi, letisztult megjelenés, gyors használat.
- Gombközpontú modulok + opcionális ChatGPT-szerű chat.
- Később: testreszabás (light/dark, téma), motiváló UI-elemek.

## 2) PDF Tools (core)
- Merge / split / oldalak törlése / átrendezése / forgatás / tömörítés.
- Konvertálás: PDF ↔ DOCX/PPTX/XLSX, képek ↔ PDF, PDF → képek.
- Extra: közvetlen PDF-szerkesztés (szöveg/grafika hozzáadás), űrlapkitöltés.
- Védelem: jelszó, titkosítás, digitális aláírás.
- Haladó (később): OCR (scannelt PDF → kereshető), vízjel/oldalszám, metaadat szerkesztés/törlés,
  2 PDF összehasonlítása, redakció (érzékeny adatok kitakarása).
- UX: “1 kattintás” eszközök, külön gombok: Összefűzés, Szétbontás, Konvertálás, stb.

## 3) AI tutor + learning analytics (később)
- Flashcard/kvíz/összefoglaló/teszt generálás feltöltött jegyzetekből és dokumentumokból.
- Részletes magyarázat, lépésről-lépésre megoldás, egyszerűbb nyelvre fordítás.
- Haladáskövetés: erősségek/gyengeségek, személyre szabott gyakorlási terv.
- Spaced repetition / aktív felidézés: gyakrabban kérdez arra, ami bizonytalan.
- Feladattípusok: feleletválasztós, kitöltős, nyílt végű, párbeszédes “virtuális tanár”.
- Opcionális: számítás + grafikon/diagram generálás (Wolfram-jellegű integráció).

## 4) Szövegszerkesztő (később)
- Word/Docs-szintű WYSIWYG: címsorok, félkövér/dőlt, listák, képek, táblázatok.
- Import/export: docx/odt/pdf.
- AI segítség: helyesírás/nyelvtan, stílusjavítás, átfogalmazás, összefoglalás.

## 5) PPT generálás (később)
- Automatikus prezentáció jegyzetből/PDF-ből (akár videó alapú anyagból is).
- Strukturálás diákra, bullet-pointok, képek/ábrák.
- Sablon/stílus választás.
- Többnyelvű PPT generálás.

## 6) Platform + fájlok (később)
- Platformfüggetlenség (Android/iOS/Windows/Mac), PWA/multiplatform opció.
- Szinkron eszközök között (jegyzet, feladat, naptár, tanulási állapot).
- Széles formátumtámogatás és egységes fájlkezelés.

## 7) Fordítás + beszéd→szöveg (később)
- AI fordító modul (dokumentumok + chat tartalom).
- Speech-to-text előadásokhoz (jogi/etikai figyelmeztetéssel: engedély).
